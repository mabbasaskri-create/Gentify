Returns & Exchanges

We make returns easy. Here's everything you need to know.

Our Return Policy

  • ✓ 30-day return window from the date of delivery
  • ✓ Items must be unworn, unwashed, and in original packaging
  • ✓ Free return shipping on all domestic orders
  • ✓ Refunds processed within 5-7 business days after receiving the item
  • ✓ Exchanges are processed immediately upon return scan
  • ✓ Final sale items are not eligible for return

How to Return

  • 1 Log into your account and go to My Orders
  • 2 Select the order and click Return Item
  • 3 Choose the item(s) and reason for return
  • 4 Print the prepaid return label
  • 5 Drop the package at any courier location
